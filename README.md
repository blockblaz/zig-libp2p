A minimal pure zig libp2p library which is intended to be used as a drop-in replacement of zeam's libp2p-glue rust wrapper (https://github.com/blockblaz/zeam/tree/main/rust/libp2p-glue).
- The library will be compliant with zig 0.16.0 specification
- it will use https://github.com/ch4r10t33r/zquic for QUIC
